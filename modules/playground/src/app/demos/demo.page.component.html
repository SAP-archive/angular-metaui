MetaUI Demo

Simple example showing how to generate a Form from the domain object.

Every MetaUI application needs to have defined for their domain object a rule file (.oss), in this case it is src/playground/rules/CarRentalRequest.oss

To play around you can visit src/playground/rules/demo/ to change above file to see how you can shape the page. Every time you change the file please run: 

 npm run oss-user
